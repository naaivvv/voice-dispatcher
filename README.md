# AI Voice Dispatcher System

A real-time, AI-powered voice dispatcher backend. This system acts as a bridge between delivery drivers on voice calls and your database, using an AI agent (LangChain + LLM) to handle intent recognition, route updates, and ETA management, while streaming human-like synthesized voice back to the driver (ElevenLabs).

## Features

- **Real-Time Voice Orchestration:** Full duplex WebSocket communication.
- **Interruption Handling ("Barge-in"):** Instantly cancels TTS streaming when the driver interrupts the AI.
- **Autonomous Tool Calling:** Powered by LangChain, the AI agent can read from and write to a Supabase PostgreSQL database dynamically based on the conversation.
- **Latency Monitored:** Built-in session performance tracking metrics accessible via `/health` and `/sessions` REST endpoints.
- **Modular LLM Architecture:** Easily swap out OpenAI for Groq, Google Gemini, or local models (via Ollama) to save on API costs.

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Copy `.env.example` to `.env` and configure your API keys:
   ```bash
   cp .env.example .env
   ```
   *Required Keys: Supabase URL/Service Role, OpenAI API Key, ElevenLabs API Key.*

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   npm start
   ```
